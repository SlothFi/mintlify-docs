A minor update focusing on mobile release, bug fixes, new mutations and improvements for login system.

## New Updates
- **Mobile Release: **The game is now available to download on iOS and Android, for Apartment and Monster token holders. For instructions, join our discord.
- Added Support for multiple notification types.
- Fixed login issues for user accounts without second name.
- Added support to zoom-in and zoom-out in preferences.
- **Daycare: **Users can now access the inventory within the daycare.
- **Classic Sailorman Bundle** is here, find cool items such as EXP booster, Visible potential booster and Eye of Athena in a single package!
- **Login Improvements: **Improved security with support for protection from attacks, as well as persistent logins added. No need to login again and again whenever you open the game anymore!
- **New Mutations: **4 new mutations have been added to the game, can you guess which ones?
- **Guilds: **Few fixes added focusing on member joining/leaving, as well as user interface updates for guilds. Now accessible at the bottom left of your screen.
- **PvP Leagues: **Added support for random battle locations to spice up PvP battles.
- **Moves &amp; Abilities Changes:**
Tidal Toxin (Ability added)
Acid Dribble (Move added)
Steam Cloud (Move added)
Toxic Cloud (Move added)
Weight Wave (Move added)
Fire Turn (Move added)
- Added support for Emojis in chat.
- Level limits for Monster Spawns near Broken bridge have been increased to 14+.
- Added new daily task focusing on PvP Leagues.
- Bug related to Genesis Toy claims has been fixed.
- Bug related to NFT monsters summary not opening has been fixed.
- Added follower sprites for 4 monsters.
