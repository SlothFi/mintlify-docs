A minor update focusing on structural improvements, feature additions and preparation for Spellborne Wrapped.

## New Updates
- A new variant is in the wild, find the Festive Pandoro Variant in Sunshine Trail.
- Santa's Gift Bundle is live.
- New Preferences: You can now edit timer for wild battles, as well as hide monsters in the wild via account preferences.
- Task Board: Added structural improvements to allow for more complex tasks (to be added in future), added support for real time counters in task notifications for progress.
- PvP Leagues: Added bots for matches when a match is not found for 10 seconds.
- Squad Presets: Improved user flow, players can now switch presets within inventory. Users with Stellar Pass subscription get access to 2 extra presets.
- Major performance improvements on Safari (iOS) should reduce crashes.
- PvP Leagues: Rewards for Era 1 are live, leaderboards are also live.
- Added move checks to help reduce missing move issues.
- Double XP Week is live until 13th December 2024.
- Improved design for multiple user interfaces such as Task Board, Daycare and more.
- Fixed bugs related to multiple moves and abilities.
- Added support for spectating your friends in battles.
- Fixes issues related to multiple achievements.

## Known Bugs
- Certain monster abilities might not work correctly.
- Certain monster breeds might not be available as followers.
- PvP leaderboard might be missing some data.

## Coming Next
- Spellborne Wrapped, will go live in December 2024.
- Wheel of Fortune: Golden Ticket, is targeted to go live in December 2024.
- Mobile Builds (iOS and Android) will go into closed testing for holders in December 2024.
- Professions are targeted to go live in December 2024.
- Settlers Glory: Apartment Staking, targeted to go live in December, 2024.
- Guilds: Base Systems, targeted to go live in December, 2024.
- Winter Season begins on 29th December 2024.
