A minor update focusing on new events, Abstract expansion and bug fixes.

## New Updates
- Spellborne is officially live on iOS and Android, make sure to download the app for easy access, in-game notifications and other benefits.
- Treasures of Celestia event is here, collect rare monsters such as Abster, Ronke and YelloKet. With added support for multiple partner tokens as payment methods.
- **5 days left for Summer: **Stellar Pass has been extended till 23rd August 2025.
- Payment System in the game has been fully revamped to support improved transaction handling and allow better transparency.
- Fishing is live in game, please read previous changelog for a detailed writeup.
- Abstract Global Wallet support has been added to the game, players can now vote for Spellborne via a weekly mission to support the game on Abstract Portal.
- Bug fixes for Settlers Glory subscription, sound effects and missing monster sprites.
- Performance updates for users using iOS and Android builds.

## Known Bugs

- Certain monster breeds might not be available as followers.
