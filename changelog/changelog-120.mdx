A minor update focusing on day care v2, bug fixes for multiple features and overall game improvements.

## Data Migration

Players may find some of their monsters missing in the daycare, this is primarily due to the active data migration that is currently going on.

The missing monsters will appear on their own in the daycare once data as been migrated, players are not required to do anything.

## New Updates
- **Daycare:** The daycare system has gone under a massive upgrade with the following features being added in this update:
- Multi-select monsters (Usable for tags and freeing into wild)
- Search filter (Searches for nicknames and breed names)
- Special filters for Soul Exchange monsters.
- Sorting based on specific innate stats.
- Design change for the monster rows in daycare.
- Monster summary has been converted into a popup.
- Support for Search templates (limited to 5 for non-Stellar Pass subscribers)
- Query improvements for faster search.
- **Guilds**: Fixed multiple bugs related to guilds and added new features:
- Added Guild Search
- Added Sub-Guild support (A/B testing, full version will be live in next update)
- Fixed a bug where users could request to join multiple guilds, causing them to switch from one guild to another on approval.
- **Stellar Pass**: The Pyramid Texts is live.
- **Double XP Week** is live.
- Global in-game chat now syncs with Spellborne discord.
- Sacrifice move now becomes active as soon as the monster is low on stamina for any one of the available moves.
- **Moves &amp; Abilities Changes:**
Zapchance (Fixed description)
Loyalty (Fixed description)
Carved Rock (Fixed description)
Hard Shell (Fixed description)
Toxic Shed (Chance to poison reduced to 50%)
