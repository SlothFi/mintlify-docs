---
title: Get Bazaar Stats
api: "GET /api/bazaar/stats"
description: Get bazaar overview statistics
---
<Note>Requires `x-api-token` header.</Note>

### Response Fields

| Field | Description |
|-------|-------------|
| totalTrades | Total trade count |
| totalVolume | All-time volume |
| activeListings | Current active listings |
| activeOffers | Current active offers |
| latestTrades | Recent trade array |
| topVolumeItems | Highest volume items |
| priceChanges24h | 24-hour price changes |
| priceChanges7d | 7-day price changes |
